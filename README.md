# aqarobhur.com — مشروع عقار أبحر

Starter بسيط لموقع عقارات أبحر الشمالية (App Router) — جاهز للرفع على GitHub والنشر على Vercel.

## تشغيل محلياً (اختياري)
```bash
npm install
npm run dev
```
